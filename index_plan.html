    工具發布計畫

    切換工具的開放狀態。左側網站地圖、已開放／不顯示清單會即時同步根目錄 sitemap.txt（發布清單）；sitemap.xml 為搜尋引擎用，應由 sitemap.txt 產生，而非反向同步。

    運作方式與限制
    • file:// 開啟：第一次請按「連結 sitemap.txt」，選專案根目錄的檔案，之後切換會自動寫入並每 2 秒監聽異動。
    • plan 伺服器：npm run plan:serve 可自動讀寫並更新 sitemap-published.js。
    • 發布清單：sitemap.txt 決定哪些工具出現在首頁與左側地圖；plan 頁的已開放／不顯示清單以此為準。
    • SEO 地圖：sitemap.xml 給搜尋引擎爬蟲用，請執行 node scripts/generate-sitemap.mjs 依 sitemap.txt 產生（勿手動當成發布清單）。
    • 即時同步：左側網站地圖、已開放／不顯示清單會跟著 sitemap.txt 更新；你在 VS Code 改檔也會自動反映。
    • 預覽：點 slug 或工具名稱可在新分頁開啟頁面。
