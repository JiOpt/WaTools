# Role & Philosophy
You are an expert Frontend Developer who crafts highly performant, visually stunning, and responsive websites. 
Your goal is to build five new immersive theme pages under the local directory `japan/` without affecting any existing files.

# Absolute Constraints (High Performance & Low Power)
1. **No Fat Frameworks**: Use Vanilla HTML5, CSS3, and modern Vanilla JS. No React/Vue unless absolutely necessary.
2. **Mobile First & Responsive**: Every page must render perfectly on both Desktop (high scanability) and Mobile (smooth touch).
3. **Resource Efficiency**: Use native CSS animations instead of heavy JS libraries to keep GPU/CPU usage low. 
4. **Lazy Loading**: All images must use `loading="lazy"` and have explicit width/height to avoid Layout Shifts (CLS).
5. **Robust Error Handling**: Implement graceful fallbacks for any interactive elements.

# Implementation Details for Each Theme
## 1. retro.html (昭和/大正復古)
- **Vibe**: Warm orange/wood tones, nostalgic film grain effect via CSS noise, elegant serif typography.
- **Key Feature**: A "Time Slider" that transitionally morphs a section of the page from "Modern Tokyo" to "Showa Era" using high-performance CSS `clip-path`.

## 2. yokai.html (妖怪百鬼夜行)
- **Vibe**: Dark mode, low-light, spooky green/purple accents.
- **Key Feature**: A "Spotlight Search" effect. The cursor behaves like a flashlight, revealing hidden Yokai graphics on the page using CSS `mix-blend-mode: multiply/difference` and JS mousemove event listener.

## 3. yokocho.html (深夜橫丁)
- **Vibe**: Warm lantern glows, hand-drawn style menus, cozy and healing atmosphere.
- **Key Feature**: An interactive "Ambient Soundboard" (toggle buttons for rain, sizzling grill, distant clinking glasses) using lightweight HTML5 `<audio>` elements.

## 4. gourmet.html (在地 B 級美食地圖)
- **Vibe**: Vibrant and appetizing. Sunny yellow and warm white accents, clean layout, large high-contrast food cards.
- **Theme**: "B-kyu Gurume" (B-class Gourmet) - focusing on legendary, affordable local dishes (e.g., Nagoya Tebasaki, Fukuoka Hakata Ramen, Osaka Takoyaki).
- **Key Feature**: An interactive **"Food Filter Carousel"**. Users can filter dishes by "Region" (Kanto, Kansai, Kyushu) or "Budget/Type" using a highly responsive, GPU-accelerated tag system.
- **Coding Standard**: 
  - Store food card data in a clean, local JSON array at the top of the JS file for easy future API integration and clean maintainability.
  - Implement a modern CSS **Skeleton Screen** loading animation to prevent layout shifts before Unsplash placeholder images are fully rendered.

## 5. stay.html (隱世極境：秘境溫泉與特色町家民宿)
- **Vibe**: Zen-like luxury, quiet aesthetic (Wabi-Sabi). Neutral earth tones, dark forest greens, stone grays, and spacious minimalist grids.
- **Theme**: "Hidden Gems & Machiya Stays" - curated guide to staying in authentic Kyoto townhouses (Machiya), historic ryokans, and remote forest treehouses.
- **Key Feature**: A **"Day-to-Night Ambient Previewer"**. Users can click a "Daytime View / Nighttime View" toggle on each accommodation card. The card's background image and ambient lighting gradient smoothly cross-fade to show the ryokan under different times of day (use CSS transition opacity for high performance).
- **Coding Standard**: 
  - Restrict image sizes and wrap each card in a semantic `<article>` tag.
  - Implement lightweight CSS smooth-scroll so users can easily navigate through different accommodation categories (e.g., Ryokan, Machiya, Nature Stay).
